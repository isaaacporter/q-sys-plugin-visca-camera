# VISCA Camera
QSC Q-SYS plugin for RS232/IP control of VISCA cameras using the standard VISCA or VISCA over IP protocols. This plugin allows you to mix and match command sets and protocols. 

## Command Set
* VISCA
* VISCA over IP

## Connection Types
* RS232
* UDP
* TCP
